# Linux Bash Basics

## Objective

Learn how to communicate with the Linux operating system using the Bash shell by executing commands, generating output, performing basic calculations, and managing the terminal environment.

## Scenario

As part of a cybersecurity lab, I practiced using the Bash shell to execute commands and interpret their output. This exercise introduced the fundamentals of interacting with Linux through the command line, an essential skill for cybersecurity professionals.

## Tools Used

- Linux
- Bash

## Commands Used

### Generate Output

```bash
echo "hello"
echo "Ngozi Onyenwenu"
```

### Perform Calculations

```bash
expr 32 - 8
expr 3500 \* 12
```

### Clear the Terminal

```bash
clear
```

## Skills Demonstrated

- Linux Command Line
- Bash Shell Fundamentals
- Command Input and Output
- Basic Terminal Navigation
- Integer Calculations with Bash

## Key Takeaways

- Used the `echo` command to display text in the terminal.
- Used the `expr` command to perform basic arithmetic calculations.
- Used the `clear` command to reset the terminal workspace.
- Developed an understanding of how the Bash shell accepts commands and returns output.

## Outcome

Successfully practiced fundamental Bash commands and gained hands-on experience interacting with the Linux operating system through the command line. These foundational skills are essential for Linux administration and cybersecurity tasks.
