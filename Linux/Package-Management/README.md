# Linux Package Management for Security Tools

## Objective

Use the Advanced Package Tool (APT) to install, verify, remove, and manage cybersecurity tools in a Debian-based Linux environment.

## Scenario

As part of a cybersecurity lab, I used the APT package manager to install, verify, remove, and reinstall security monitoring tools. This exercise demonstrated how Linux administrators and security analysts manage software required for monitoring and securing systems.

## Tools Used

- Linux
- Bash
- APT
- sudo
- Suricata
- tcpdump

## Commands Used

### Verify APT

```bash
apt
```

### Install Suricata

```bash
sudo apt install suricata
```

### Verify Installation

```bash
suricata
```

### Remove Suricata

```bash
sudo apt remove suricata
```

### Install tcpdump

```bash
sudo apt install tcpdump
```

### List Installed Packages

```bash
apt list --installed
```

### Reinstall Suricata

```bash
sudo apt install suricata
```
