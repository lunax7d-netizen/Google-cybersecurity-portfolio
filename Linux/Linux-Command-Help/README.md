# Linux Command Help

## Objective

Use Linux help commands to learn more about commands, options, and functionality in the Bash shell.

## Scenario

As a security analyst, it is important to know how to find information about Linux commands when you do not remember how they work. Linux provides built-in help tools that allow you to discover commands and understand their options.

## Tools Used

- Linux
- Bash
- man
- whatis
- apropos

## Commands Used

### Get a short description of a command

```bash
whatis cat
