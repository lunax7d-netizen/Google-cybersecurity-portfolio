# Linux File System Navigation

## Objective

Develop foundational Linux command-line skills by navigating the file system, locating files, and reading file contents using Bash commands.

## Scenario

As part of a cybersecurity lab, I practiced navigating the Linux file system to locate directories, examine files, and retrieve information. These skills are important for security analysts when investigating systems, reviewing files, and analyzing logs.

## Tools Used

- Linux
- Bash

## Commands Used

### Display Current Directory

```bash
pwd
```

### List Directory Contents

```bash
ls
```

### Change Directory

```bash
cd /home/analyst/reports
cd /home/analyst/reports/users
cd /home/analyst/logs
```

### Read File Contents

```bash
cat Q1_added_users.txt
```

### View First Lines of a File

```bash
head server_logs.txt
```

## Skills Demonstrated

- Linux File System Navigation
- Bash Command Line Usage
- Directory Navigation
- File Reading and Analysis
- Log File Review
- Security Investigation Fundamentals

## Key Takeaways

- Used `pwd` to identify the current working directory.
- Used `ls` to view files and directories.
- Used `cd` to navigate through Linux directories.
- Used `cat` to read file contents.
- Used `head` to review log files.

## Outcome

Successfully navigated the Linux file system and used Bash commands to locate, read, and analyze files. This lab strengthened my Linux skills used in cybersecurity investigations.
