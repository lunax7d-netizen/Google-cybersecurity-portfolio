
# Linux Package Management for Security Tools

## Objective

Use the Advanced Package Tool (APT) to install, remove and verify cybersecurity applications in a Debian-based Linux environment.

## Scenario

As a security analyst, I was required to install and manage security tools using Linux package management.

## Tools Used

- Linux
- Bash
- APT
- sudo
- Suricata
- tcpdump

## Commands Used

### Verify APT

```bash
apt
```

### Install Suricata

```bash
sudo apt install suricata
```

### Verify Installation

```bash
suricata
```

### Remove Suricata

```bash
sudo apt remove suricata
```

### Install tcpdump

```bash
sudo apt install tcpdump
```

### List Installed Packages

```bash
apt list --installed
```

### Reinstall Suricata

```bash
sudo apt install suricata
```

Successfully managed the installation, removal and verification of security monitoring tools using APT.
