# Perform a SQL Query

## Objective

Use SQL queries to retrieve information from databases, select specific columns, return complete tables, and organize results using the ORDER BY keyword.

## Scenario

As a security analyst, it is important to know how to retrieve and analyze information stored in databases. In this lab, SQL queries were used to examine employee device information and investigate login activity for possible security issues.

The information was retrieved from the organization database using the machines and log_in_attempts tables.

## Tools Used

- SQL
- MariaDB
- Linux Terminal
- SELECT
- FROM
- ORDER BY
