# Assign Python Variables

## Objective

Learn how to create and assign Python variables, store different types of information, and use variables to organize data for cybersecurity tasks.

## Scenario

As a security analyst, Python variables can be used to store and manage information such as usernames, login attempts, IP addresses, and security alerts. In this lab, Python variables were created and assigned values to practice storing and working with data.

## Tools Used

- Python
- Jupyter Notebook
- Variables
- Strings
- Integers
- Lists
- Boolean Values
- print()

## Python Code

```python
# Assigning a username variable
username = "analyst"

# Assigning a login attempts variable
login_attempts = 5

# Assigning an IP address variable
ip_address = "192.168.1.25"

# Assigning an account status variable
account_locked = False

# Assigning a list of suspicious IP addresses
suspicious_ips = ["192.168.1.10", "10.0.0.5", "172.16.0.8"]

# Display variables
print(username)
print(login_attempts)
print(ip_address)
print(account_locked)
print(suspicious_ips)
```

## Takeaways

- Learned how to create and assign variables in Python.
- Practiced storing different data types such as strings, integers, lists, and Boolean values.
- Learned how variables help organize and manage information in cybersecurity tasks.
- Practiced using `print()` to display stored data.
- Built a foundation for using Python to automate security-related tasks.
